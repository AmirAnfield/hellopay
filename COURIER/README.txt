Ce dossier contient l'ensemble des fichiers nécessaires pour implémenter un générateur automatique de bulletins de paie dans le cadre du projet HELLOPAY.

Contenu :
- Bulletin_Paie_Rubriques.xlsx : Tableur listant toutes les lignes possibles d’un bulletin de paie avec formules, taux, conditions, etc.
- Guide_Calcul_Bulletin.txt : Explication complète du calcul du bulletin de salaire du brut au net.
- Modele_Base_Donnees.txt : Description de la structure de base de données recommandée.
- Logique_Algorithmique.txt : Étapes de calcul automatisé d’un bulletin, de la saisie à la génération.
